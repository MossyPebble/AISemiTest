result.csv는 다음과 같은 형식.
index.csv에 저장된 index와 result.csv의 각 행의 순서가 같음.
result.csv의 각 데이터는 줄바꿈으로 구분됨.
줄바꿈으로 구분된 result.csv의 각 데이터는 2차원 배열 csv이므로, 줄바꿈으로 구분된 데이터를 서로 분리해 열면 2차원 배열이 됨.
